使用 Hugo 搭建静态博客

2025年1月17日 阅读时间：1 分钟
Xqcherry
技术
Hugo • 静态网站 • 博客 • 教程

Hugo 是一个用 Go 语言编写的静态网站生成器，以其极快的构建速度而闻名。本文将介绍如何使用 Hugo 搭建一个静态博客。

为什么选择 Hugo？ 链接到标题

  1. 速度极快：Hugo 的构建速度是其他静态网站生成器的 10-100 倍
  2. 简单易用：使用 Markdown 编写内容，无需数据库
  3. 功能丰富：支持多语言、主题、短代码等
  4. 部署方便：生成的静态文件可以部署到任何地方

安装 Hugo 链接到标题

Windows 链接到标题 

# 使用 Chocolatey
choco install hugo-extended

# 或使用 Scoop
scoop install hugo-extended

macOS 链接到标题 

brew install hugo

Linux 链接到标题 

sudo apt-get install hugo

创建新站点 链接到标题 

# 创建新站点
hugo new site my-blog

# 进入目录
cd my-blog

# 添加主题（以 hugo-coder 为例）
git submodule add https://github.com/luizdepra/hugo-coder.git themes/hugo-coder

配置站点 链接到标题

编辑 hugo.toml 文件：

baseURL = "https://your-domain.com"
title = "我的博客"
theme = "hugo-coder"
languageCode = "zh-cn"
defaultContentLanguage = "zh-cn"

[params]
  author = "你的名字"
  description = "博客描述"

创建第一篇文章 链接到标题 

hugo new posts/我的第一篇文章.md

编辑文章内容：

+++
title = "我的第一篇文章"
date = 2025-01-15
+++

这是文章内容...

本地预览 链接到标题 

hugo server

访问 http://localhost:1313 查看效果。

构建静态文件 链接到标题 

hugo

生成的静态文件在 public/ 目录中。

部署 链接到标题

可以将 public/ 目录部署到：

  • Vercel：连接 GitHub 仓库自动部署
  • Netlify：拖拽 public/ 文件夹即可
  • GitHub Pages：推送到 gh-pages 分支
  • 自己的服务器：上传 public/ 目录内容

总结 链接到标题

Hugo 是一个强大且易用的静态网站生成器，非常适合搭建个人博客。通过简单的配置和 Markdown 编写，就能快速创建一个美观的博客网站。
